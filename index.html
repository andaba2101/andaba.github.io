TripleTen · SQL · 04/2026

Análisis de embudo y retención para MercadoLibre

Diseñé un análisis de embudo multietapa en SQL mediante CTEs para detectar los principales puntos de fuga dentro del proceso de conversión de usuarios. Segmenté resultados por país, dispositivo y fuente de tráfico para identificar caídas relevantes y estructurar reportes útiles para toma de decisiones.

También analicé la retención de usuarios por cohortes, países y procesos de compra, detectando patrones de activación y recurrencia. A partir de esos hallazgos propuse mejoras accionables orientadas a optimizar conversión y fidelización con apoyo de data storytelling.

TripleTen · Python · 05/2026

Análisis urbano y económico para American Development Bank

Integré y depuré datos de TomTom Traffic Index y OECD Cities para construir una base analítica única de aproximadamente 800.000 registros centrada en el año 2024. Con esta base calculé indicadores agregados por ciudad y evalué la relación entre congestión vehicular y PIB per cápita.

Elaboré visualizaciones en Python con Jupyter para comparar desempeño urbano y económico, e identifiqué ciudades con mejores y peores combinaciones entre movilidad y productividad. El análisis permitió formular conclusiones orientadas a inversión en transporte y priorización de políticas públicas en LATAM.
